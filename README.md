# Wavo

A minimal iOS voice memo recorder. Record audio, preview the waveform, and copy the recording (with an optional message body) to the clipboard for sharing.

## Features

- Record audio with a live animated waveform
- Playback with a progress overlay on the waveform
- Rename recordings before copying
- Compose a reusable message body that ships alongside the audio when copying to clipboard

## Requirements

- iOS 15+
- Xcode 14+

## Getting Started

Open `Wavo.xcodeproj` in Xcode, select a simulator or device, and press Run (⌘R).
